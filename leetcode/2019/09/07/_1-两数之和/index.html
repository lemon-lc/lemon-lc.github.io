# 1. 两数之和

# 题目描述

给定一个整数数组 nums  和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。 示例:

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]

# 解法

# 解法一（暴力解决法）

  • 解题思路： 暴力法就是直接遍历 x , 继续遍历 x 之后的 y , 判断 x + y 是否等于 target。

    假设 nums = [1, 2, 7, 11, 15], target = 9

    nums[0] + nums[1] = 3 != target，继续循环，


    **...**

    **nums[0] + nums[4] = 16 != target，继续循环，**

    **nums[1] + nums[2] = 9 = target，终止循环，返回 [1, 2]。**


  • 代码实现：

/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function(nums, target) {
  const length = nums.length;
  for (let i = 0; i < length - 1; i++) {
    for (let j = i + 1; j < length; j++) {
      if (nums[i] + nums[j] == target) {
        return [i, j];
      }
    }
  }
};

# 解法二（一遍哈希表）

  • 解题思路： 新建一个 Map，对数组进行遍历，假设遍历的是 x，res = target - x，再判断一下 Map 中是否有 res，没有的话把当前的值作为 key，索引作为 value 存入 Map 中，有的话则终止循环，返回 Map 的 value 以及当前的索引值。

    当然也可以使用一个对象进行存储数据，只需要注意一下对象的 key 值是字符串。

    假设 nums = [1, 2, 7, 11, 15], target = 9

    创建一个 Map，开始循环 target-nums[0] = 8, 判断 Map 中是否有键值为 8 的数据 不存在键值为 8 的值，则把 nums[0]和索引 0 存入 Map 中, 继续循环 ...
    target-nums[2] = 2, Map 中存在键值为 2 的数据，返回 [Map[2], 2] => [1, 2]

  • 代码实现：

var twoSum = function(nums, target) {
  const map = new Map();
  for (let i = 0; i < nums.length; i++) {
    const num = nums[i];
    const res = target - num;
    const value = map.get(res);
    if (value !== undefined) {
      return [value, i];
    }
    map.set(num, i);
  }
};

# 解法三

根据解法二我们也可以使用另外一种方案，每次检测一下一遍数组中是否存在差值。

使用 indexOf 的执行速度没有使用哈希表执行速度快

var twoSum = function(nums, target) {
  for (let i = 0; i < nums.length; i++) {
    const num = nums[i];
    const res = target - num;
    const index = nums.indexOf(res);
    // 判断数组中是否差值 且不是当前值
    if (index >= 0 && index !== i) {
      return [i, index];
    }
  }
};
